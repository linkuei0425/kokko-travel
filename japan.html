一、 2026 日本入境最新規定

Visit Japan Web
入境與海關申報

目前日本全面推廣線上通關。請在班機起飛前至 VJW 網站填寫資料，取得「外國人入境記錄」與「海關申報」的 QR Code。截圖保存在手機內，過海關與稅關時直接刷條碼，大幅節省排隊時間。

免簽證待遇
90天觀光免簽

持中華民國（台灣）護照的旅客，只要護照效期在預計離開日本時仍有效，即可享有90天免簽證的待遇，無需事前申請任何簽證電子許可。

同行家人同行申報
家庭共用便利設定

若與嬰幼兒或長輩同行，可在 VJW 系統中設定為「同行家人」，海關申報部分只需由一位代表出示 QR Code 即可，非常方便。

二、 日本自由行必備神級 APP 與數位工具

日本的地鐵路網與生活消費高度數位化，強烈建議出發前將以下工具下載或加入手機主畫面：

Web App 專屬攻略

沖繩通 Okinawa Pass

免下載 APP！自駕導航、海灘秘境與國際通美食指南

Web App 專屬攻略

大阪通 Osaka Pass

環球影城、心齋橋美食與關西廣域交通一次掌握

Web App 專屬攻略

福岡通 Fukuoka Pass

屋台美食、太宰府與九州鐵道周遊全攻略

Google Maps 找路與營業時間

在日本，Google Maps 絕對是最好用的地圖軟體，步行導航極為精準，且能即時查看餐廳營業時間與評價。搭配「儲存清單」功能，行前規劃超方便。

乘換案內 (Japan Transit Planner) 地鐵防走失神器

東京與大阪的地鐵網路極為複雜。這款 APP 支援繁體中文，能精確列出發車時間、月台番號，甚至會貼心提示「在哪個車廂下車最靠近換乘通道或電梯」，避免在新宿或梅田車站迷路！

GO Taxi / Uber 叫車必備

日本計程車雖然較貴，但在走累時是救星。GO 是日本本土覆蓋率最高的叫車 App；若不想多註冊，使用原本的 Uber App 在日本也能直接叫到合作的在地計程車，並直接用綁定的信用卡扣款。

Payke 藥妝掃碼翻譯

在日本逛藥妝店或超市時的神器！只要掃描商品背面的條碼，就能直接在手機上顯示繁體中文的商品資訊、成分與使用方法，不怕買錯東西。

Tabelog (食べログ) 在地美食評鑑

日本最具公信力的餐廳評論網站。分數只要超過 3.5 分通常就是極具水準的美店，超過 3.8 分則是神級名店。找尋在地人愛吃的餐廳用它就對了。

三、 行前實用準備與交通

  • 網卡與機場交通預訂

    抵達機場後，順暢連接網路與搭上特快車是首要任務。出發前建議先透過旅遊平台買好 eSIM / 實體網卡，以及 Skyliner (東京) / Haruka (關西) 等機場快線票券，落地即可直接掃 QR Code 劃位上車。

  • 支付與數位交通卡 (Suica / ICOCA)

    日本目前信用卡接受度已達九成，但仍建議準備適量現金以應付小店或神社賽錢。強烈推薦 iPhone 用戶出發前在「錢包 (Apple Wallet)」中加入 數位 Suica (西瓜卡) 或 ICOCA，綁定信用卡加值後，不論是搭地鐵、超商買宵夜、或是投自動販賣機，手機感應即可秒速完成。

  • 電壓與轉接頭規格

    日本電壓為 100V，插座為兩孔扁型（A 型），與台灣的兩孔插頭完全相同！台灣的手機、行動電源充電器可直接插入使用，無需帶轉接頭。但需注意，若您的設備是三孔（有接地線），則需自備三轉二轉接頭。

四、 親子同遊特別注意事項

推車與無障礙動線 (多功能廁所)

日本車站普遍較舊，許多月台的電梯位在特定車廂位置。推嬰兒車時，請善用車站內的案內圖找尋電梯標誌。另外，日本大型商場或車站的「多功能廁所 (誰でもトイレ)」空間極大，非常適合推車直接推入，並多附有尿布台。

居酒屋與餐廳年齡限制

在日本，部分帶有菸味的傳統居酒屋，或高檔的預約制餐廳（如無菜單料理 Omakase），會婉拒未滿一定年齡的兒童入內以維護環境。帶小孩用餐建議優先選擇家庭餐廳 (Family Restaurant)、連鎖定食店或百貨公司樓上的美食街，對兒童絕對友善且常提供兒童餐具。

氣候與常備藥品準備

日本冬季或初春時，室外寒冷但室內（電車、商場）暖氣非常強烈。強烈建議採「洋蔥式穿搭」，進室內才不會熱到流汗吹風感冒。日本氣候極為乾燥，請務必幫幼童準備滋潤乳液。雖然日本藥妝店林立，但仍建議備妥慣用的兒童專用退燒、腸胃等常備藥。

五、 日本在地文化與生活習慣

2026 全新免稅 (Tax-Free) 制度

日本免稅制度迎來重大變革！為防範轉賣，目前改採「先付後退」機制。旅客在市區購物需先支付含稅全額，並由店家登錄資訊。離境時，請預留充足時間於機場的免稅終端機掃描護照及查驗商品，系統才會將消費稅退還至您的信用卡或電子支付中。過往嚴格的「免稅密封袋」限制也隨之解綁，但離境前務必妥善保管商品以供查驗。

地方住宿稅與入湯稅

入住東京、大阪、京都、福岡等觀光城市時，除了訂房網上已結清的房費外，飯店櫃檯會在您辦理入住 (Check-in) 時，依當地法規額外收取每人每晚 100 至 500 日圓不等的「住宿稅」；若入住溫泉旅館則需額外繳交「入湯稅」。這筆規費多數飯店僅收現金，請記得多準備一些零錢在身上。

無小費文化與付錢托盤

日本絕對沒有收取小費的習慣，留下零錢服務生會追出來還你。另外，在櫃檯結帳付現時，請將鈔票與硬幣放在櫃檯上的「小托盤」內，避免直接將錢遞給店員的手，這是當地相當重視的禮貌細節。

街頭無垃圾桶與邊走邊吃

日本街頭幾乎找不到垃圾桶，自動販賣機旁的回收桶「僅供丟棄寶特瓶/鐵罐」。出門請隨身準備小塑膠袋，將垃圾帶回飯店丟棄。此外，日本文化中認為「邊走邊吃」是不禮貌的，在攤販買了小吃，請在店門口或指定的角落站著吃完再離開。

洗手間衛生紙規則

日本的衛生紙大多具備水溶性，如廁後請直接將使用過的衛生紙丟入馬桶中沖掉。廁所內若有放置小垃圾桶，通常是專門用來丟棄生理用品的，請勿將擦拭過的衛生紙丟入其中。
